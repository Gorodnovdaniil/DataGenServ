# DataGen - Генератор синтетических данных

Система генерации синтетических данных на основе имитационного моделирования.

## Особенности

- Генерация случайных величин различных распределений (равномерное, экспоненциальное, гамма, нормальное, Пуассона, треугольное)
- Все генераторы реализованы вручную на основе равномерного распределения U(0,1)
- RAG-движок для поиска информации в лекциях по имитационному моделированию
- Веб-интерфейс на Streamlit
- Поддержка seed для воспроизводимости результатов

## Быстрый старт с Docker Compose

### 1. Запуск системы одной командой

```bash
docker-compose up --build
```

После запуска откройте браузер и перейдите по адресу: **http://localhost:8501**

### 2. Остановка системы

```bash
docker-compose down
```

## Локальный запуск (без Docker)

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Запуск приложения

```bash
streamlit run front/app.py
```

Откройте браузер и перейдите по адресу: **http://localhost:8501**

## Доступные распределения

- **Uniform** - равномерное распределение U(a, b)
- **Exponential** - экспоненциальное распределение с параметром λ
- **Gamma** - гамма-распределение с параметрами α и β
- **Normal** - нормальное распределение N(μ, σ²)
- **Poisson** - распределение Пуассона с параметром λ
- **Triangular** - треугольное распределение с параметрами a, b, c

## Интеграция с n8n

Для подключения LLM через n8n:

1. Настройте API endpoint для вашего экземпляра DataGen
2. Используйте вебхуки в n8n для отправки запросов
3. Подключите ваш предпочтительный LLM-провайдер (OpenAI, Anthropic, etc.)
4. Настройте обработку ответов для RAG-функциональности

## Разработка

### Добавление нового распределения

1. Реализуйте метод генерации в `backend/generator.py`
2. Добавьте информацию о распределении в `backend/main.py` (метод `get_distribution_info`)
3. Обновите UI в `front/app.py` для поддержки новых параметров
